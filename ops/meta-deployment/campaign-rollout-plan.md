## Week 1 — Foundation

- [ ] Verify all gates
- [ ] Clean tracking
- [ ] Launch Campaign 3 (Brand Awareness) at minimum budget
- [ ] Launch Campaign 1 (GLP-1) with 2-3 ad variations at test budget
- [ ] Launch Campaign 2 (Sofwave) if Gate 4 passes
- [ ] Verify forms, links, and response handling
- [ ] Establish baseline metrics (label as PROVISIONAL)
- [ ] Begin daily DM/comment response cadence

**Do not expect conversions to be trusted yet.**

## Week 2 — Signal Collection

- [ ] Cut obvious losers ($50+ spend, zero conversions)
- [ ] Keep only credible creative variations
- [ ] Fix any landing page mismatches discovered
- [ ] Build retargeting audience pools
- [ ] Launch Campaign 4 (Men's Waitlist) if capacity exists

**Do not scale. Do not declare winners from small samples.**

## Week 3 — Early Optimization

- [ ] Test stronger hooks on winning ad sets
- [ ] Test shorter forms if lead quality is acceptable
- [ ] Segment creative by service intent
- [ ] Improve DM speed and booking flow
- [ ] First weekly report with honest confidence labels

## Week 4 — First Scale Decision

- [ ] Scale only what has trusted signal (30+ conversions, 14+ days)
- [ ] Pause fantasy campaigns
- [ ] Produce next month's creative based on actual top hooks
- [ ] Update service launch matrix
- [ ] First monthly report

## Month 2+ — Ongoing

- Increase budgets on proven campaigns (max 20% per adjustment, 5-7 day wait)
- Expand creative testing
- Launch Campaign 5 (Peptides) only if Gate 2 explicitly passes
- Evaluate Sofwave ROI vs. lease cost (~$2,037/month)
- Build full-funnel retargeting sequences
- Begin attribution refinement

## Ad Set Structure Per Campaign

### Campaign 1 — GLP-1
- **Ad Set A:** Broad targeting, Washington State
- **Ad Set B:** Retarget website visitors (non-converters)
- **Ad Set C:** Retarget form abandoners
- **Ad Set D:** Lookalike/modeled audiences (when data available)
- **Ads per set:** 3-5 compliant variations
- **Placements:** Start broad, narrow with data

### Campaign 2 — Sofwave
- **Ad Set A:** Local broad (15mi Renton)
- **Ad Set B:** Retarget engagers
- **Ad Set C:** Cross-sell audiences (if compliant)

### Campaign 3 — Brand Awareness
- **Ad Set A:** Local broad
- **Purpose:** Fill top of funnel, create retargetable audiences

### Campaign 4 — Men's Waitlist
- **Ad Set A:** Broad within WA
- **Rule:** ALL messaging must remain waitlist-only

## Creative Testing Rule

Test one major variable at a time:
- hook
- thumbnail / cover
- headline
- CTA
- landing page angle

Do not test five new ideas at once and then pretend you learned something precise.
