## 12. BATCH ORDERING PROTOCOL

### Weekly Olympia Order — Every Monday

1. All HRT refills due this week
2. All skincare refills
3. All NAD+ / Sermorelin injectable refills
4. All sexual health product orders
5. All supply restocking (syringes, swabs)
6. **Consolidate shipping** — single shipment saves on freight

### Weekly Hallandale Order — Every Monday

1. All Bella #1 SR refills
2. All troche refills (Sermorelin, NAD+)
3. Any items not available from Olympia

### QualiphyRx — Per-Patient, Ongoing

1. All tirzepatide prescriptions (90-day auto-refill)
2. All semaglutide prescriptions (90-day auto-refill)
3. CJC/Ipamorelin, Epithalon, MOTS-C, Tesamorelin orders
4. Default setting: **"Provider Selects Best Price"** always routes to Greenwich

## 13. COST GUARD RAILS

| Rule | Rationale |
|------|-----------|
| **NEVER** route GLP-1s (tirzepatide/semaglutide) to Hallandale or Olympia | QualiphyRx Greenwich is always cheapest |
| **NEVER** order NAD+ or Sermorelin injectables from QualiphyRx | Olympia is 50–70% cheaper |
| **NEVER** order HRT from Hallandale when Olympia carries it | Olympia beats Hallandale pricing on almost every HRT product |
| **ALWAYS** check if Olympia carries a product before ordering from Hallandale | Hallandale is the backup, not the default |
| **NEVER** switch a patient's pharmacy mid-prescription cycle | Wait until next refill to change source |
