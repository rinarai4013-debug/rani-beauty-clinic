## 7. Data Gaps & Recommendations

### Blocked/Missing:
1. **Square** — Chrome extension blocks access. Need to either:
   - Whitelist `app.squareup.com` in browser extension settings
   - Generate a Personal Access Token at `developer.squareup.com` for API access
   - This would give us payment transaction data with amounts

2. **BECU Banking** — Chrome extension blocks access. Options:
   - Use Plaid integration (API routes already built) to pull bank transactions
   - Or manually export bank statements as CSV

3. **Mangomint** — No API, no export. Options:
   - Enable Mangomint Webhooks ($50/mo add-on) for real-time booking data
   - Manually export client list if Mangomint adds this feature
   - Use the booking widget embed data as a proxy

### Recommended Next Steps:
1. **Square API Token** — Go to `developer.squareup.com`, create a Personal Access Token, add to `.env` as `SQUARE_ACCESS_TOKEN`
2. **Plaid for BECU** — Use existing Plaid API routes to connect BECU checking account
3. **Mangomint Webhooks** — Enable $50/mo webhook add-on for real-time booking sync to Airtable
4. **Airtable Sync** — Import Jotform clients + Cherry financing data into Airtable Clients table
