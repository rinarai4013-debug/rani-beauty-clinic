## PHOTO SPECS (for whoever is shooting)

```
Format:     JPEG or WebP
Resolution: 1920x1080 minimum (landscape) or 1080x1350 (portrait)
File size:  Keep under 500KB per image (compress with TinyPNG or Squoosh)
Naming:     lowercase, dashes, descriptive
            e.g., "hydrafacial-treatment-closeup.jpg"
            Need 2 per service: one primary (1.jpg) + one hover (2.jpg)
Lighting:   Warm, soft, professional — no harsh overhead fluorescents
Background: Clean, uncluttered, on-brand (navy/gold/cream tones work great)
```

## VIDEO SPECS

```
Format:     MP4 (H.264)
Resolution: 1080p minimum, 4K preferred
Duration:   15-60 seconds for web, longer cuts for YouTube
File size:  Under 10MB for autoplay heroes, larger OK for click-to-play
Aspect:     16:9 for website, also capture 9:16 vertical for reels/TikTok
Audio:      Optional — most web videos autoplay muted
```
