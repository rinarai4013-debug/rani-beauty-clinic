# ElevenLabs Voice Clone Recording Script
# Record all sections below in one continuous session
# Total time: ~25-30 minutes
# Environment: Quiet room, phone or USB mic, natural speaking voice

---

## SETUP INSTRUCTIONS

1. Use your phone Voice Memos app or a USB microphone on your laptop
2. Quiet room — no music, no AC hum, no background noise
3. Speak naturally — exactly how you talk to clients, not "radio voice"
4. If you mess up a line, pause 3 seconds and re-read it — ElevenLabs handles it
5. Record as ONE continuous file (WAV or M4A, not MP3)
6. Upload to: https://elevenlabs.io → Voices → Add Voice → Instant Voice Cloning
7. Name the voice: "Rani"
8. Use the Voice ID in your Vapi assistant config

---
