# Local Environment Setup

This project requires Node.js 20.x (see `.nvmrc-note`).

## Prerequisites
- Node.js 20.x installed
- npm available in PATH

## Install
```bash
npm install
```

## Common Commands
```bash
npm run dev
npm run lint
npm run typecheck
npm test
```

## Notes
- If `node` or `npm` are missing, install Node.js 20.x via your preferred toolchain (nvm, fnm, asdf) and re-run the commands above.
