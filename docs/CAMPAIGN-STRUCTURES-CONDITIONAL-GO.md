## Compliance Rules (All Conditional GO Campaigns)

**MANDATORY — these rules override all other guidance:**

1. **NO prescription drug names** in keywords, ad copy, headlines, or descriptions
   - Blocked terms: semaglutide, tirzepatide, liraglutide, Ozempic, Wegovy, Mounjaro, Zepbound
   - Blocked terms: testosterone, estradiol, progesterone, DHEA, pregnenolone
   - Blocked terms: BPC-157, PT-141, CJC-1295, Ipamorelin, Sermorelin (by name in ads)
2. **NO remarketing** on weight-loss campaigns (Google sensitive interest category)
3. **Generic service-level language only** — describe the service category, not the medication
4. **All copy requires owner approval** before going live (per Commander OS governance)
5. **Landing pages must NOT have drug names in the URL** that Google crawls as the final URL

## Conditional GO — Compliance Monitoring Protocol

**Weekly checks:**
- [ ] Search term report: flag any drug-name queries triggering ads
- [ ] Add drug-name queries as negative keywords immediately
- [ ] Verify no ad copy contains restricted terms
- [ ] Check for Google policy warnings or ad disapprovals
- [ ] Confirm Campaign 8 is not serving remarketing (should be Search only)
- [ ] Review landing page for any new drug-name content added by other team members

**Escalation triggers:**
- Any ad disapproval mentioning "Restricted drug terms" → pause campaign, review, fix
- Any policy warning on the account → same-day review
- CPC exceeding cap by >20% → reduce bid or pause low-performing keywords
- CTR < 1.5% after 2 weeks → review ad copy and keyword relevance
